File Integrity checks found files for splunk

1, 2018 Article

File Integrity checks found 1 files that did not match the system-provided manifest. See splunkd.log for details.

갑자기 다가온 알수 없는 오류…먼가가 안맞단다…

원인을 찾아보자

Solution1.

SHC에서

https://[SPLUNK SHC]:8089/services/server/status//installed-file-integrity

Solution2.

cd $SPLUNK_HOME/bin

./splunk validate files

문제가 되는 파일을 확인할수 있다.

이유는 다양하니 해결책은 알맞게….

./splunk restart

그리고 SHC에서 스플렁크를 재시작하면 해당 문구가 사라지는걸 볼수 있다.
